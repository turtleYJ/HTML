1. 기본 속성 선택자

기본 선택자 뒤에 []를 사용하여 속성과 속성값을 사용한다. 

        선택자[속성=속성값] {
            스타일 속성: 값;
            스타일 속성: 값;
            ...
        }

2. 자손 선택자와 후손 선택자

자손 : 바로 하위 요소들 / 후손 : 하위 요소들 전부 

        1) 자손 선택자
            a>b {
                스타일 속성: 값;
                스타일 속성: 값;
                ...
            }
        2) 후손 선택자
            a b {
                스타일 속성: 값;
                스타일 속성: 값;
                ...
            }

    • 동위 선택자

    동위 관계(같은 레벨)에서 뒤에 위치한 태그를 선택할 때 사용한다. 

            1) a 선택자 뒤에 있는 b요소 한 개 선택
            a+b {
                스타일 속성: 값;
                스타일 속성: 값;
                ...
            }

        2) a 선택자 뒤에 있는 모든 b 요소 선택
            a~b {
                스타일 속성: 값;
                스타일 속성: 값;
                ...
            }

    4. 반응 선택자

    사용자의 음직임에 따라 달라지는 선택자 

            1) 사용자가 클릭하는 요소에 스타일 적용
            선택자:active {
                스타일 속성: 값;
                스타일 속성: 값;
                ...
            }
        2) 사용자가 마우스를 올려놓는 요소에 스타일 적용
            선택자:hover {
                스타일 속성: 값;
                스타일 속성: 값;
                ...
            }

    5. 상태 선택자

    요소의 상태에 따라 선택되는 선택자 

            1) 체크된 상태의 태그
            선택자:checked {
                스타일 속성: 값;
                스타일 속성: 값;
                ...
            }
    사과 바나나 복숭아 

            2) 초점이 맞추어진 input 태그의 스타일 변경
            선택자:focus {
                스타일 속성: 값;
                스타일 속성: 값;
                ...
            }
